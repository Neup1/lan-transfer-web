# 局域网文件传输（LAN Transfer）

一个面向日常办公的局域网简易传文件工具。
- 两台机器在同一子网内

## 具体功能
- 浏览器即可上传/下载文件
- 支持大文件与断点续传
- 支持拖拽上传、多文件上传
- 支持中英文切换
- 支持服务器端切换上传根目录
- 登录口令默认隐藏显示，支持显示/隐藏切换

## 环境要求
- Windows / macOS / Linux
- Python 3.9+
- 无需安装第三方依赖

## 快速开始

### 第一步：在服务器电脑启动
```powershell
python lan_transfer.py
```

启动后终端会显示：
- 访问地址（例如 `http://192.168.1.20:9000/`）
- 6 位通行口令（大写字母+数字）

### 第二步：在任意客户端电脑打开浏览器
访问上面的地址，输入口令登录即可传文件。

## 页面使用说明

### 上传文件
- 点击 `选择文件` 或直接拖拽到上传区域
- 点击 `上传`
- 勾选“同名文件断点续传”时，重复上传不会从头开始

### 新建文件夹
- 可让不同的客户端机器自己在根目录下创建并分别上传到不同子目录
- 例如：`部门A/`、`部门B/`、`设备01/`

### 切换上传根目录（仅服务器本机）
在服务器侧左栏 `上传根目录`：
- 可手动输入目录后点击 `应用`
- 也可点 `选择文件夹` 直接从系统窗口选择

## 常见问题

### 端口被占用
可直接自动处理（kill掉原端口程序）：
```powershell
python lan_transfer.py serve --port-conflict kill
```

### 登录失败
- 确认输入的是本次启动时终端显示的口令
- 注意字母大小写

## 可选命令（进阶）

自定义共享目录：
```powershell
python lan_transfer.py serve --root "D:\LAN_SHARE"
```

固定端口：
```powershell
python lan_transfer.py serve --port 9000
```
